# WeiOS Roadmap

## Week 1: Project Skeleton

- Create monorepo.
- Add NestJS API skeleton.
- Add Flutter desktop shell.
- Add Next.js web shell.
- Add shared TypeScript models.
- Add PostgreSQL, Redis, and MinIO local infrastructure.

## Week 2: Project Hub

- Project list and detail API.
- Project status, priority, current state, next actions, and risks.
- Seed projects: Planfit, iTMS, WeiOS, Flutter Finance, AI Video, IELTS, Investment.

## Week 3: Memory Engine v0

- Manual memory creation.
- Project-linked memory list.
- Tags and search.
- Decision memory and risk memory types.
- Communication paste import.

## Week 4: AI Team v0

- Agent role registry.
- Provider configuration.
- Project context loading.
- Summary and next-action generation.

## Week 5: Security Center v0

- Risk classifier.
- Pending approvals.
- Audit log UI and API.
- Red-action blocking.

## Week 6: Communication Import v0

- Manual WeChat and DingTalk paste flow.
- Extract facts, decisions, tasks, and risks.
- Write approved results back to Project Hub and Memory Engine.
